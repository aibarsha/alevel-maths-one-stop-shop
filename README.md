## Suggested revision routine

1. Pick one unit and one topic.
2. Complete the Maths Genie topic questions first.
3. Use Corbett for an extra explanation/worksheet where available.
4. Use PMT/Pearson past papers once the topic feels secure.
5. Record mistakes and redo similar questions one week later.
