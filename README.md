## Suggested revision routine

1. Pick one unit and one topic.
2. Complete the Maths Genie topic questions first.
3. Use the Corbett link for extra practice or foundation recap.
4. Move to unit past papers once the topic is secure.
5. Record mistakes in a revision log and redo similar questions one week later.
